# Retired studio web UI (June 2026)

`index.html`, `app.js`, `styles.css` were the front-end for the old
`scripts/banner_studio.py` http.server. The rebuild replaced the interactive
server with a maintainable CLI (`scripts/fai_banner.py`) plus a static
contact-sheet generator (`scripts/fai_contact.py`). These files are kept for
reference only and are not served by anything.

The logo SVGs that remain in `studio/` (`logomark-*.svg`, `acronym-black.svg`)
are brand assets, not part of the retired UI, and are intentionally left in
place.
